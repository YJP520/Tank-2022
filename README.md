#### **pygame游戏引擎的安装**
- pip安装
    - windows + R --> cmd --> 命令行输入 pip install 模块名 == 版本号
- pycharm中安装
    - File --> setting --> project --> Project Interpreter --> 右键 + install --> 搜索框输入pygame -- 下方installPackage
